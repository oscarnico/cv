# 💻 CV - Óscar Nicolás

Portfolio personal desarrollado con React + Vite.

🔗 https://oscarnico.github.io/cv/

---

## 🚀 Tecnologías

- React
- Vite
- JavaScript
- CSS
- GitHub Pages

---

## 🧪 Desarrollo local

```bash
npm install
npm run dev 
```
Abrir en: http://localhost:5173/

---

## 📦 Build

```bash
npm run build
```

Genera la carpeta /dist

## 🌐 Deploy (GitHub Pages)
```bash
npm run deploy
```
## 🔁 Workflow
Subir cambios al repo
```bash
git add .
git commit -m "mensaje"
git push origin main
```

Actualizar la web:
```bash
npm run deploy
```
